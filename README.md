# Description
Plugin SambungKata diambil dari kosakata Bahasa Indonesia untuk PaperMC. If you are have a question, please make a [issues](https://github.com/Kylan1940/PaperMC-SambungKata/issues/new)

# Features
>- Timer
>- Mistakes
>- Points
>- 10.000++ words
>- Multiplayer and Singleplayer
>- Configuration

# Installation
1. Download the [plugin]()
2. Put plugin to **plugins** file
3. Restart the server

# Commands
| Command | Description |
|---|---|
| /sambung | Memulai room |
| /sambung join <namaHost> | Join ke room sesuai nama Host yang memulai /sambung |
| /sambung leave | Meninggalkan room |
| /sambung list | Melihat seluruh pemain dalam room |
| /sambung start | Memulai pertandingan |
